# Module 1 Assignment: Actionable Usage of Excel

This repository contains my first assignment for the Data Analysis Course. 

## 📌 Tasks Completed:
* Created a dataset of 10 students with subjects, marks, and attendance.
* Formatted the worksheet (Center alignment, Percentage format).
* Used **Absolute Referencing** (`$G$1`) to calculate bonus marks.
* Utilized **Flash Fill** to generate unique Student Codes.
* Applied **Conditional Formatting** based on criteria (Green for >80, Yellow for 50-79, Red for <50).
* Sorted the dataset by Marks (Highest to Lowest) and practiced Filtering.
* Implemented **Freeze Panes** for better readability.
* Set up Page Layout, Margins, Header/Footer, and configured the sheet to **Fit on One Page** for printing.
